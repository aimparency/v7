# Three-Minute Demo Script

## 0:00-0:25 - Problem

"Git tells us what changed. It does not tell an AI agent why the change matters,
which goals depend on it, or whether the implementation fulfilled the original
intent. That context is increasingly scattered across chats and ticket tools."

## 0:25-0:55 - Intent Graph

Open Aimparency on the current repository. Select a high-level aim, show its
children and dependencies, then show value, cost, and status.

"Aimparency stores an editable model of intent next to the repository. Humans
remain the authority, while agents can read the same structure."

## 0:55-1:30 - Agent Execution

Show prioritized aim discovery and open the loop panel.

"The agent does not receive a random task. It orients in the graph, sees why the
aim matters, and gets only the capability packs this loop needs."

Show coding, experiments, and code-intelligence packs.

## 1:30-2:05 - Evidence and Dependencies

Show a structured experiment and a waiting dependency.

"Real work includes uncertainty and actions that require people or external
institutions. The loop records hypotheses and evidence, delegates when needed,
and resumes when the dependency is resolved."

## 2:05-2:35 - Git-Native Proof

Open `.bowman/` and a Git diff.

"The important state is ordinary project data: local, diffable, reviewable, and
not locked inside one model provider's conversation history."

## 2:35-3:00 - Ask

"We are looking for three design partners or a small angel bridge. EUR 15,000
funds six months of focused validation: real teams, paid use, reliable
intent-to-code provenance, and a clear next financing decision."
