# Official Source Notes

These are concise research notes, not copies of the source pages. Last verified
July 18, 2026.

## Event Overview

URL: https://openai.devpost.com/

- OpenAI Build Week is an online public event centered on building with Codex
  and GPT-5.6.
- Deadline shown: July 21, 2026 at 5:00 PM PDT.
- Event page showed approximately 41,000 participants at verification time.
- Total prize pool: USD 100,000.
- Four tracks: Apps for Your Life, Work and Productivity, Developer Tools, and
  Education.
- Developer Tools expressly covers testing, DevOps, agentic workflows, and
  security.
- The event page requires a working project, project description, sub-three-
  minute public YouTube demo, repository, README guidance, and `/feedback`
  Codex session ID.
- Judging dimensions: technological implementation, design, potential impact,
  and quality of the idea.

## Official Rules

URL: https://openai.devpost.com/rules

- Registration runs July 9-21; submission work is scoped to July 13-21.
- Existing projects may enter if meaningfully extended with Codex and/or
  GPT-5.6 after the submission period starts.
- Only the new work in a pre-existing project is evaluated.
- Evidence distinguishing old and new work can include timestamped Codex
  sessions and dated commit history.
- The demo must include audio and explain both what was built and how Codex and
  GPT-5.6 were used.
- A public repository needs relevant licensing. A private repository must be
  shared with the judging addresses in the rules.
- Developer tools need installation instructions, supported platforms, and an
  accessible testing route that does not require judges to rebuild everything.
- Judges may evaluate only the submitted description, images, and video rather
  than testing the project.
- Stage-one screening checks viability and fit. The four stage-two criteria are
  equally weighted.
- Developer Tools awards: one USD 15,000 first prize and one USD 10,000 second
  prize.

## Verification Rule

Before final submission, re-open both official URLs and check for updates. The
event rules state that the official rules and website override summaries or
plugin output.
