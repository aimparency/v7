## Part A — Proposal Information

### Abstract (max 2000 chars)

Aimparency is a Human-AI Cooperation Engine: a local-first software platform that replaces opaque agentic AI with an explicit, editable graph of goals ("aims"). As autonomous agents execute code, research, and multi-step workflows, humans lose visibility into intent, priorities, and strategy. Aimparency provides the missing coordination layer — a git-native will-model where value flows from high-level aims and costs bubble up from actions, enabling intelligent prioritization while preserving human sovereignty.

We have a working prototype: graph UI, MCP agent integration, multi-session agent execution (Claude, Gemini, Codex), and semantic search over aims. We target technical founders and power users first, then position as infrastructure ("Will-UI") for the agentic AI ecosystem. Business model: SaaS subscription + compute margin. We request EIC support to scale R&D, achieve TRL 8, and establish a European standard for accountable autonomous systems.

## Innovation (Section 5 — 12-page body)

### 1. The Innovation

**Problem:** Agentic AI systems (coding agents, operators, multi-step automations) execute increasingly complex action chains without exposing goals, trade-offs, or strategic intent. Humans cannot inspect, guide, or resolve conflicts without breaking autonomy. Existing tools model tasks, not intentions.

**Our solution:** A graph-based will-model where every AI intention is an explicit node with support relationships, intrinsic value, and cost. Humans collaborate on *why* the AI acts; agents execute on *what* the graph specifies. State lives in `.bowman` directories inside user repositories — git-versioned, auditable, agent-readable.

**Breakthrough elements:**
1. Aims as coordination primitive (not task lists or hidden chains)
2. Value/cost flow for automatic prioritization
3. MCP-native agent orchestration across sessions
4. Reflection pattern for cross-session learning
5. Economic autonomy layer (sustainability ratio)

**TRL:** 4–5 today (validated prototype). Target TRL 8 in 24 months.

### 2. State of the Art & Differentiation

| Competitor | Approach | Gap |
|------------|----------|-----|
| OpenAI Operator / ChatGPT Agent | Hidden multi-step chains | No strategic goal layer |
| Manus | General autonomous agent | Opaque execution |
| Cursor / Copilot | Code domain only | No life/project goals |
| Motion / Lindy | Task automation | No will-model / git audit |

Aimparency is the **legitimacy and coordination layer** — not a foundation model, not a task board. We build on OpenRouter/multi-model infrastructure.

### 3. Technology & IP

- Open-source graph format (`.bowman`) for adoption
- Proprietary SaaS orchestration, hosting, enterprise features
- No blocking third-party patents identified
- FTO: standard OSS stack (Node.js, Vue, tRPC)

### 4. Development roadmap (24 months)

**M1–6:** Production agent loop (discover→plan→execute→verify), reflection engine  
**M7–12:** Cross-session memory, multi-agent specialization, voice interface MVP
