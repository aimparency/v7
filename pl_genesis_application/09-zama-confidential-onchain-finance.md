# Zama: Confidential Onchain Finance

## Challenge Information
- **Sponsor:** Zama
- **Prize Pool:** $5,000
- **Category:** Privacy & FHE

## Challenge Overview

Public blockchains are transparent by default, exposing balances, strategies, and user activity to the public. This level of transparency has hindered the adoption of onchain finance by institutions, DeFi protocols and everyday users.

With the Zama Protocol, powered by Fully Homomorphic Encryption (FHE), computation can run directly on encrypted data, enabling confidential and verifiable onchain finance.

Your challenge is to design and prototype applications that bring confidentiality on public blockchain, and build financial systems where sensitive data remains encrypted at all times.

Developers can write confidential smart contracts using Solidity and familiar tooling. By using encrypted integer (euint) data types, developers can easily define which parts of smart contracts remain private and specify access rules that determine who can decrypt specific data directly within the smart contract logic.
