# Autonomy Project Structure

This directory contains documentation, scripts, and logic related to the **Economic Autonomy** of the agent.

## Contents
- `metrics.md`: Definition of success metrics (Revenue, Efficiency, Ratio).
- `scripts/`: Scripts to calculate and report these metrics.
- `reports/`: Generated reports on autonomy status.

## Purpose
To quantify and track the agent's progress towards infinite self-sustainment.
