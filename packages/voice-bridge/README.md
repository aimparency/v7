# Voice Bridge

`voice-bridge` is an optional development package that exposes voice-related
services for Aimparency. It is not part of the default local-first startup
path and is currently lower priority than the main browser UI, backend, MCP,
and wrapped-agent runtime.

Use the repository root scripts for the main application:

- `npm run dev`
- `npm run start`

Only use this package directly if you are working on voice-specific features.
