# Mobile

This directory contains an experimental Android client for Aimparency.

It is not part of the main local-first open source release path. A new user does not need it to run Aimparency locally.

Current role:

- exploratory mobile and voice-interface work
- Android app scaffold built with Kotlin and Jetpack Compose
- separate from the main browser UI in `packages/frontend`

If voice work resumes, this package is one place where that exploration may continue. For now, the primary supported interface is the local browser UI.
